# StackFlow Frontend

React + TypeScript frontend for StackFlow, a social micro-tipping app on Stacks.

## Local development

Requirements:
- Node.js 20+
- npm 10+

Run locally:

```sh
cd frontend
npm install
npm run dev
```

Build for production:

```sh
npm run build
npm run preview
```

Run tests:

```sh
npm test
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vitest
